usage: collector-to-emulator [-h] [-v] [-i PATH] [-t DIR] [-n NAME] [-s PATH]
                             [JSONL]

convert kafka-collector output into kafka-emulator config

positional arguments:
  JSONL                 JSONL file (if stdin is a TTY and -i omitted)

options:
  -h, --help            show this help message and exit
  -v, --version         show program's version number and exit
  -i PATH, --input PATH
                        JSONL file (ignored when stdin is piped)
  -t DIR, --template-dir DIR
                        template output directory (default: templates/)
  -n NAME, --name NAME  scenario name in generated YAML (default: Unnamed)
  -s PATH, --scenario PATH
                        scenario YAML path when stdout is a TTY (default:
                        scenario.yaml)
